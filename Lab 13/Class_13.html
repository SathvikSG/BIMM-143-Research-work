Q7.

The arr.ind causes the which() function to return both the row and column indices. Since meancounts[,1:2] is a matrix with two columns (control.mean and treated.mean), this allows us to see exactly which genes (rows) and which conditions (columns) contain zero expression values.

While calling unique() ensures we don’t count any row twice if it has zero entries on both the samples.
